Polyfills, GamePad API et Accéléromètre

  • Les polyfills utilisent l'accéléromètre pour implémenter l'API de WebVR pour les téléphones mobiles
  • Configuration pour optimiser la performance
  • CardBoardVRDisplay défini les capacités du téléphone mobile
  • https://github.com/immersive-web/webvr-polyfill
  • Il faut utiliser les outils de base (three.js, GamePad API) ou des librairies plus haut niveau comme a-frame pour offrir une expérience desktop
  • Utiliser en combinaison avec aframe ou react-vr
