Composants ReactVR

  • View: Un peu comme div, hérité de React Native
  • Image: Comme img, hérité de React Native
  • Text: Comme p, hérité de React Native
  • Pano: Permet l'affichage d'un panorama, un peu comme a-sky de a-frame
  • VRButton: Permet d'intéragir avec la scène 3D
  • AppRegistry.registerComponent: Enregistrement du composant root. Initialiser une appplication React VR est comme React Native plutôt que React.
