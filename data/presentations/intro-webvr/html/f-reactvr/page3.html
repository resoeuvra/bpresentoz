Débuter avec ReactVR

  • npm install -g react-vr-cli: Installer l'outil de ligne de commande react vr
  • react-vr init VotreProjet: Initialiser votre projet
  • npm install --save webvr-polyfill: Pour ne pas oublier Safari
  • Modifier le fichier client.js pour ajouter le polyfill
  • Dans le répertoire npm start: Permet d'intéragir avec la scène 3D
