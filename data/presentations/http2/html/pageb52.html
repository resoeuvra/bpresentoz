Lexique

  • CDN : Content Delivery Network / Réseau de livraison de contenu
  • RTT : Round trip time : Temps de réponse d'un point réseau à un autre (incluant le retour)
  • Frame / Trame, Ensemble encapsulé de données sur un réseau
  • Stream / Signal / Flux : Ensemble logique de trames formant par exemple un fichier, une image, etc.
