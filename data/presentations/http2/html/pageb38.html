Apache 2 et Ubuntu 15.10

  • Apache 2 inclus mod_http2 de 2.4.17 (nous sommes à 2.4.18 au moment d'écrire ces lignes)
  • Apache 2 2.4.17 sera inclus dans la prochaine version de Ubuntu 16.04 fin avril
  • Il existe un repo avec la dernière version de Apache (ppa:ondrej)
  • sudo add-apt-repository ppa:ondrej/apache2
  • sudo apt-get update
  • sudo apt-get install apache2
  • sudo a2enmod http2
  • sudo a2enmod ssl
  • Cette installation inclus tout ce qui est nécessaire
