Nginx CentOS 7

  • Nginx prends en charge une bonne partie de HTTP2 (sauf server push) depuis novembre 2015
  • La version 1.9.12 (au moment d'écrire ces lignes - mars 2016) n'est pas une version stable
  • La version stable 1.10.1 est disponible depuis juin 2016
  • CentOS n'a jamais les dernières versions des packages (pour des bonnes raisons)
  • APLN nécessite openssl 1.0.2 - CentOS a 1.0.1
  • Le plus simple est de compiler nginx statiquement lié à openssl 1.0.2 (installer 1.0.2 sur CentOS risque de vous causer du trouble)
  • openssl 1.1.0 est maintenant disponible (juin 2016)
  • Utiliser les repo "remi" "epel" pour installer les dépendances (nombreuses pcre-devel)
  • Télécharger la source de nginx, et télécharger la source de openssl 1.0.2g dans le même répertoire
  • ./configure avec --with-openssl=./openssl-1.0.2g
