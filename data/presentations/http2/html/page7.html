Problèmes avec HTTP 1.0 et HTTP 1.1

  • Le pipelining HTTP 1.1 n'aide pas vraiment le blocage en tête de file
  • Des requêtes malicieuses peuvent causer des denis de service
  • N'est pas pris en charge par défaut sur les navigateurs dû à de nombreux problèmes de serveurs non conformes
  • HTTP 1.1 ne prends pas en charge le multiplexage (plusieurs signaux en même temps)
