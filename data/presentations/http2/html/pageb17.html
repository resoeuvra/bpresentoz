Autres bonnes pratiques

  • Utiliser un CDN (pour diminuer la latence)
  • Minimisation de code
  • Limiter les méta données qui se répètent dans les en-têtes
  • Minimiser les redirections
  • Réutiliser les connexions TCP (Keep-alive)
  • Bien utiliser les techniques de caching client
  • Limiter le pre-téléchargement (balancer le pré-téléchargement et la congestion)
