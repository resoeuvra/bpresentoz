Configurer http2 avec IIS sous Windows 10

  • Pris en charge par défaut - il n'y a rien à faire...
  • Il faut toutefois modifier le format de nos certificats SSL de letsencrypt
  • openssl pkcs12 -inkey privkey1.pem -in fullchain1.pem -export -out iis_pfx.pfx
  • Et configurer SSL
    1. Ouvrir IIS Manager
    2. Aller à Nom de la machine, Section IIS, Server Certificates
    3. Dans le menu de droite : Action -> Import
    4. Choisir le fichier pfx créé précédemment et le mettre dans le store Certificate hosting
    5. Aller ensuite avec le menu gauche sur votre site Web
    6. Sur le menu de droite, choisir Action -> Bindings
    7. Faire Add -> Choisir votre hostname, port 443 et associer votre certificat
  • Microsoft offre une API pour le server Push HttpResponse.PushPromise
